## 0. Hook · 30초

"내일 아침 7시에 너에게 이런 뉴스레터가 도착한다"를 화면에 띄우고 시작합니다.

거꾸로 본질이 깔리면 1~5가 자동으로 "그걸 만들기 위한 단계"로 묶입니다.

<Callout type="tip">
**투표 결과 통합 멘트** (오프닝 끝에): "1번 4명, 3번 4명으로 두 갈래가 나왔는데, 같이 보니까 두 안이 같은 substrate를 다른 면에서 묻는 거예요. 1번('뭘 모르는지')이 매일 돌아가려면 3번 저장소가 자연스럽게 따라옵니다. 그래서 1번을 앵커로 잡고, 3번은 그 앵커를 떠받치는 substrate로 같이 만듭니다. 2번('실행으로')은 마지막에 '다음 한 줄' 위젯으로 흘려보내고, 풀스케일은 다음 주 후보로 빼둘게요."
</Callout>

## 1. 통증 회상 · 7분

본인 PKM의 통증 1줄.

> "지금 PKM 어디가 막혀 있나?"

- 세컨드 브레인, 옵시디언, 노션, 폴더, 메모앱 — 뭐든 좋습니다
- 침묵, 본인 노트
- 채팅에는 한 줄로만 (전원, 30초)

<Callout type="warning">
"잘 안 쓴다", "지식이 쌓이기만 한다" 같은 추상 X. **손에 잡힌 통증 1줄**. 막힌 장면이 보이게.
</Callout>

## 2. 외부 모델 짚기 · 10분

세 가지를 각 1줄로 짚고, 본인 케이스 매칭으로 닫습니다. **강의 모드 빠지지 않게 8분 박스에 가둡니다.**

| 모델 | 한 줄 |
|---|---|
| 안드레 카파시 · LLM wiki | LLM이 가장 잘 읽도록 평문 + 링크 + 시간순으로 쌓는 저장소 |
| 김창준 · PKM | 외부화의 도구는 곧 사고의 도구다. 쓰는 게 곧 생각하는 것 |
| PKM vs PIM | PKM은 **지식**을 다루고, PIM은 **정보**를 다룬다. 정보 정리에 빠지면 지식이 안 생긴다 |

**본인 케이스 매칭**: "너는 어디에 가까워?" 한 줄 → 채팅.

## 3. 메커니즘 1장 · 15분 (메인 액티비티 ①)

오늘 라이브의 hinge입니다. 미리 그려둔 outbox 다이어그램 v0를 화면에 띄우고, 같이 수정·커스텀합니다.

→ 상세는 [outbox 메커니즘](/week2/substrate) 페이지 참조.

<Callout type="warning">
가장 자주 무너지는 지점이 여기입니다. "메커니즘"이 추상어로 남으면 4·5가 따라가 흔들립니다. **종이에 v0를 그려두고 들어가서, 라이브에선 그걸 같이 수정하는 형태**가 안전합니다.
</Callout>

## 4. 저장소 클론 · 10분

미리 만든 초안 저장소를 clone합니다. 강사가 GitHub URL을 채팅에 공유.

저장소에 포함된 것:

- `inbox/` · `outbox/` · `processed/` 3-폴더 구조
- 클로드 코드 커맨드 1~2개 (`/morning`, `/digest` 등)
- 본인 데이터 둘 위치 README

<Callout type="tip">
4번은 5번을 위한 셋업 시간입니다. 10분 안에 끝내고 바로 5번으로 넘어가세요. 셋업이 길어지면 에너지가 빠집니다.
</Callout>

## 5. 데이터 + 커맨드 · 25분 (메인 액티비티 ②)

본인 데이터 1소스를 `inbox/`에 넣고 커맨드를 돌립니다.

<Steps>
<Step title='데이터 소스 1개 픽 · 5분'>
PR 5개 · 노션 3페이지 · 노트 1폴더 중 가장 손에 잡히는 것 하나.
</Step>
<Step title='inbox/에 넣기 · 5분'>
복붙 · 다운로드 · 링크 — 어떤 방식이든. 정리 X, 그냥 넣기.
</Step>
<Step title='커맨드 1개 돌리기 · 10분'>
`/morning` 또는 `/digest`. outbox/에 1줄 메타인지 + 1줄 액션이 나오는지 확인.
</Step>
<Step title='맞다/아니다 즉시 판정 · 5분'>
"이게 진짜 내가 모르고 있던 거 맞나?" 본인이 즉시 짚기. 틀리면 프롬프트 한 번 더.
</Step>
</Steps>

## 6. 스케줄 v1 · 15분

매일 아침 7시에 뉴스레터가 도착하도록 스케줄을 겁니다.

**v1 범위 (최소)**:
- 뉴스레터 1회 · 매일 아침 7시
- 본문: 메타인지 1줄 + 다음 한 줄 액션 + 근거 인용

**v1에서 빼는 것** (다음 주 후보):
- 메타인지 지도 (전체 매핑)
- 방향 검토 위젯
- 뉴스레터 풀스케일 (요약 · 정리 · 분류)

<Callout type="warning">
6번에 5가지를 한 번에 담으면 v1이 v0.3이 됩니다. **앞 3개만** (뉴스레터 + 메타인지 1줄 + 액션 1줄) 잡고, 나머지는 다음 주로 빼두세요.
</Callout>
