# 2. 라이브 셋업 — 채널 입장과 첫 인사

> 라이브 시작 슬롯. 디스코드 본인 채널에 들어가서 본인 에이전트와 한 줄 인사하면 끝.

## 준비물

라이브에 사용할 헤르메스 에이전트는 본인 디스코드 채널에 이미 준비되어 있습니다. 본인이 따로 설치하거나 API key를 챙길 필요는 없어요.

## 1단계 — 디스코드 본인 채널 입장

라이브 시작 시간에:

1. 디스코드 챌린지 서버 접속
2. 본인 이름이 박힌 채널(예: `#agent-홍길동`)로 이동
3. 채널 안에 본인 에이전트가 봇으로 들어와 있는지 확인

## 2단계 — 첫 한 줄 대화

채널에 한 줄 던집니다. 예시:

- "안녕, 내 에이전트야. 너는 누구니?"
- "오늘 라이브에서 우리가 같이 할 일 한 줄로 정리해줘."
- "내 사용 가능한 슬래시 커맨드를 보여줘."

다음 단원([3. 내 학습 키워드를 SKILL.md로](/week4/skill))로 넘어갑니다.

## (선택) 본인 환경에 직접 설치하고 싶다면

라이브는 디스코드에 준비된 에이전트로 진행하지만, 본인 머신에 직접 헤르메스를 깔고 싶다면 두 가지 길이 있습니다. **둘 다 공식 문서 기반**이라 본인에게 편한 쪽으로 고르면 됩니다.

| 방식 | 누가 편한가 | 설치 시작점 |
|---|---|---|
| **데스크탑 앱** | macOS·Windows에서 클릭 한 번이 편한 분 | [macOS .dmg](https://hermes-assets.nousresearch.com/Hermes-Setup.dmg) · [Windows .exe](https://hermes-assets.nousresearch.com/Hermes-Setup.exe) · [공식 안내](https://hermes-agent.nousresearch.com/desktop) |
| **CLI** | 터미널·Python 환경이 편한 분 | `pip install hermes-agent && hermes postinstall` · [공식 quickstart](https://hermes-agent.nousresearch.com/docs/getting-started/quickstart) |

설치 후 provider 선택·첫 chat 검증·스킬 추가 절차는 공식 문서에 한 장으로 정리되어 있어요. 라이브 시간에 이 길로 가는 분은 본인 페이스로 따라오시고, 막히는 지점이 있으면 디스코드 본인 채널에서 함께 짚습니다. 본인 환경 이관은 [미션](/week4/mission)에서 정식 과제로 진행합니다.
