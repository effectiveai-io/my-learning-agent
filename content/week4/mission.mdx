# 🚀 미션 · 6/9 (화) 23:59 마감

> 라이브에서 깔린 v1을 **본인 환경으로 이관** + 7일 굴려본 후 회고.

## 라이브에서 v1이 완성된 상태

라이브가 끝나는 시점에 본인은 다음 두 가지를 갖춥니다:

1. 디스코드 채널에 살고 있는 본인 에이전트 + SKILL.md 1장
2. outbox 한 편 → 본인 키워드 적용 → 다음 학습 한 줄로 잇는 회로를 라이브에서 한 바퀴 굴려본 경험

운영자 인프라에서 동작 중인 상태입니다. 미션은 이걸 **본인 환경으로 이관**하고 7일 동안 굴려보는 작업입니다.

<Callout type="warning">
**운영자 인프라 운영 기간** — 라이브 후 N일 동안만 운영자 API key로 동작합니다(라이브에서 정확한 날짜 안내). 그 뒤로는 본인이 직접 헤르메스를 설치해야 메모리·스킬을 잃지 않고 이어갈 수 있습니다. **주도권 이전**이 미션의 핵심.
</Callout>

## 결과물 3가지

1. **본인 환경에 헤르메스 설치**
   - 데스크탑 앱: [macOS](https://hermes-assets.nousresearch.com/Hermes-Setup.dmg) · [Windows](https://hermes-assets.nousresearch.com/Hermes-Setup.exe)
   - CLI: `pip install hermes-agent && hermes postinstall` ([공식 quickstart](https://hermes-agent.nousresearch.com/docs/getting-started/quickstart))
2. **SKILL.md 이관** — 라이브에서 만든 SKILL.md를 본인 헤르메스에 옮기고 슬래시 커맨드 호출 확인
3. **7일 누적 후 outbox 1단락** — 매일 한 번씩 본인 키워드 스킬을 본인 outbox에 적용해서 다음 학습 한 줄을 받았는지, 그게 다음 날 outbox로 이어졌는지 1단락 회고

## 검증 1줄

본인 머신 헤르메스에서 `/{본인 커맨드명}` 한 번 입력 → 결과가 라이브에서 본 그 형태로 돌아오는가. 본인 outbox 한 편을 컨텍스트로 줬을 때 다음 학습 한 줄을 만들어주는가.

🎯 **미션의 본질** = **"본인 통제 하에 v1이 굴러가는 상태"**. 운영자 인프라에서 본인 인프라로 **주도권 이전**이 완료되어야 5회차 데모데이에서 "4주 누적 한 단어"를 펼칠 때 본인의 진짜 변화를 본인이 직접 확인할 수 있습니다.
</Callout>
