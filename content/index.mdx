## 이렇게 작동합니다

내가 쌓은 데이터를 매일 아침 1줄로 정리해서 본인에게 돌려보냅니다.

```mermaid
flowchart LR
    In["내가 쌓은 데이터<br/>PR · 노션 · 노트"]
    Proc["에이전트<br/>읽고 · 정리하고 · 골라낸다"]
    Out["매일 아침 도착<br/>메타인지 1줄<br/>다음 액션 1줄"]

    In --> Proc --> Out

    classDef anchor fill:#dee9fb,stroke:#3d6ff2,stroke-width:2px,color:#1f47b3
    classDef out fill:#dcf0e3,stroke:#3a8a5a,stroke-width:1.5px,color:#1f5a3a
    class Proc anchor
    class Out out
```

매주 라이브에서 한 조각씩 늘립니다. 1회차 프롬프트 → 2회차 저장소 → 3회차 추론 → 4회차 자율 행동. 4주 뒤 결과물은 슬라이드가 아니라 **쓸수록 나에게 맞춰 성장하는 에이전트 한 대**입니다.
